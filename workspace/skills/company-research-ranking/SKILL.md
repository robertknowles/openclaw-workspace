# Company Research Ranking Skill

## Objective
Create a process to research, rank, and document Sydney buyer's agent companies focused on investment properties.

## Steps:
1. Log research task to the ResearchPipeline sheet using the sheet-tracker skill.
2. Update the task status to "in progress" when research starts.
3. Find 3 companies (via internet or mock data).
4. Rank them on a scale of 1-10 based on:
   - Investment focus
   - Portfolio size
   - Tech adoption
5. Save the research and ranking to memory/research/companies/sydney-buyers-agents.md.
6. Register the output and update task status to complete.

## Research Methodology
- Use search engines or LinkedIn to find potential companies.
- Assess each based on criteria.
- If no internet, produce a mocked-up dataset.

## Output Format
"Company Name: [Name]"
- "Score: [number]/10"
- "Investment focus: [description]"
- "Portfolio size: [number]+ properties"
- "Tech adoption: [high/medium/low]"
- "Contact: [email]"
