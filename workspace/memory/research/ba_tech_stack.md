# BA Tech Stack Research (Feb 2026)

## Sources
- PropertyChat "Tools of a Buyers Agent 2023" thread: https://www.propertychat.com.au/community/threads/tools-of-a-buyers-agent-2023.73954/

## CRM / Project Management
| Tool | Users | Notes |
|------|-------|-------|
| **Monday.com** | Most popular (Berny, MyPropertyPro) | Free version too basic; paid version for 5-10 clients |
| **Trello** | Berny | Free - day-to-day to-do lists |

## Communication
| Tool | Users | Notes |
|------|-------|-------|
| **WhatsApp** | Universal | Client communication, agent networks for off-market deals |
| **MailChimp** | Berny | Bulk emails to agents |
| **Cellcast** | Berny | Pay-per-text bulk SMS to agents |

## Document Signing
- **DocuSign** - Standard for client agreements

## Property Data / Due Diligence
- **RP Data** - Property reports
- **Pricefinder** - Property data
- **Chrome extensions/widgets** - Price ranges on re.com.au listings (e.g., Homer/KoalaData)

## Accounting
- **Xero** - Standard

## Key Insight
Most BAs cobble together 5-7 separate tools. No unified platform exists for the full client lifecycle (brief → search → due diligence → acquisition → portfolio tracking).
